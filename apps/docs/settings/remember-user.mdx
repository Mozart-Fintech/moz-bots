---
title: Recordar usuario
icon: bookmark
---

Dirígete a la pestaña `Configuración` de tu mozbot, en la sección `General` puedes encontrar la configuración `Recordar usuario`.

Esta configuración te permite guardar el estado de la sesión de chat en el almacenamiento del navegador web del usuario. Esto significa que si responde una pregunta y luego cierra el chat, la próxima vez que lo abra, el chat estará en el mismo estado en que estaba antes.

Hay 2 opciones de almacenamiento:

- **Almacenamiento local**: El estado del chat se guardará en el navegador web del usuario. Estará disponible solo en el mismo dispositivo y navegador web.
- **Almacenamiento de sesión**: El estado del chat se guardará en el navegador web del usuario. Estará disponible solo en el mismo dispositivo y navegador web, pero se eliminará cuando el usuario cierre la pestaña actual o el navegador web.
