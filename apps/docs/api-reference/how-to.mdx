### Cómo encontrar mi `publicId`

Para la ejecución del mozbot publicado, necesitas proporcionar el ID público del mozbot disponible aquí:

<img src="/images/api/publicId.png" alt="Obtener ID de mozbot" />

### Cómo encontrar mi `workspaceId`

En el panel de tu espacio de trabajo, dirígete a `Configuración y Miembros > Espacio de trabajo > Configuración` y copia el ID del espacio de trabajo.
