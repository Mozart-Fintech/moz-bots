openapi: POST /v2/generate-upload-url
---

Los campos `presignedUrl` y `formData` pueden ser utilizados para subir el archivo al bucket de S3, directamente desde el navegador si es necesario. AquÃ­ tienes un ejemplo:

```js
// los datos contienen los campos presignedUrl y formData de la respuesta

const formData = new FormData()
Object.entries(data.formData).forEach(([key, value]) => {
  formData.append(key, value)
})
formData.append('file', file)
const upload = await fetch(data.presignedUrl, {
  method: 'POST',
  body: formData,
})
```
