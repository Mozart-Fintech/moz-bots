Las variables son marcadores de posición para contenido que luego puedes usar en cualquier parte del Mozbot. Es un concepto muy importante de entender para crear una experiencia verdaderamente personalizada para el usuario.

## Guardar una respuesta en una variable

Puedes decirle a tu bloque de entrada que guarde la respuesta en una variable y luego reutilizarla en una burbuja posterior, por ejemplo:

<Frame>

## Usar variables

Una vez que tus variables están declaradas, puedes usarlas **en cualquier parte** de tu bot. Por ejemplo, puedes mostrarlas en una burbuja de texto con la siguiente sintaxis:

`{{Mi variable}}` donde "Mi variable" es el nombre de tu variable.
