El bloque lógico de enlace a mozbot te permite ir a otro flujo de mozbot. Esto ayuda a mantener tus flujos limpios y reutilizar un flujo en múltiples lugares.

## Compartir variables entre bots

Los valores de las variables existentes se comparten automáticamente con el bot enlazado. Esto significa que si este bot enlazado contiene nombres de variables similares, se rellenarán automáticamente con los valores del bot anterior.

Ejemplo: Mi primer bot pide el nombre del usuario y lo almacena en la variable `Name`. Luego, enlazo a otro bot que muestra una variable `Name` en una burbuja de texto. Esto mostrará el nombre recogido en el primer bot.

## Fusionar respuestas

La opción Fusionar respuestas te permite fusionar las respuestas recogidas de un bot enlazado con el bot actual. Esto es útil si deseas recopilar respuestas de múltiples bots y luego enviarlas todas a la vez a una aplicación de terceros. O si simplemente deseas recopilar todas las respuestas en una tabla de resultados unificada.
