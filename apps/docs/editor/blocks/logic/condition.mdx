El bloque de Condición te permite dividir tu flujo en dos basado en una condición.

<Frame caption='Esto se puede traducir a: "Si la puntuación es mayor que 20, entonces ve por este camino. De lo contrario, ve por el otro camino."'>
  <img src="/images/blocks/logic/condition.png" alt="Condición" />
</Frame>

Una condición puede contener diferentes comparaciones que se evalúan en orden y se enlazan con un operador lógico: 'AND' o 'OR'.

## Operadores

<ResponseField name="Igual a">
  Coincidirá si el valor proporcionado es estrictamente igual al valor.
</ResponseField>

<ResponseField name="No igual a">
  Coincidirá si el valor proporcionado no es igual al valor.
</ResponseField>

<ResponseField name="Contiene">
  Coincidirá si el valor proporcionado contiene el valor. Si se proporciona una
  lista, coincidirá si la lista tiene al menos un elemento en común con el
  valor.
</ResponseField>

<ResponseField name="No contiene">
  Igual que `Contiene` pero coincidirá con el inverso.
</ResponseField>

<ResponseField name="Mayor que">
  Coincidirá si el valor proporcionado es mayor o igual que el valor.
</ResponseField>

<ResponseField name="Menor que">
  Coincidirá si el valor proporcionado es menor o igual que el valor.
</ResponseField>

<ResponseField name="Está establecido">
  Coincidirá si el valor proporcionado no es nulo o indefinido y no es una
  cadena vacía.
</ResponseField>

<ResponseField name="Está vacío">
  Coincidirá si el valor proporcionado es nulo, indefinido o una cadena vacía.
</ResponseField>

<ResponseField name="Empieza con">
  Coincidirá si el valor proporcionado comienza con el valor.
</ResponseField>

<ResponseField name="Termina con">
  Coincidirá si el valor proporcionado termina con el valor.
</ResponseField>

<ResponseField name="Coincide con regex">
  El valor debe comenzar y terminar con `/` y contener un patrón regex válido.

Ejemplo:

- `/^hola$/` coincidirá si la cadena es estrictamente igual a "hola".
- `/hola/` coincidirá si la cadena contiene "hola". Como "hola mundo".
- `/hola/i` coincidirá si la cadena contiene "hola" sin distinguir mayúsculas de minúsculas. Como "Hola mundo".
- `/[0-9]+/` coincidirá si la cadena contiene uno o más dígitos. Como "123".

</ResponseField>

<ResponseField name="No coincide con regex">
  Igual que `Coincide con regex` pero coincidirá si el valor proporcionado no
  coincide con el patrón regex.
</ResponseField>
