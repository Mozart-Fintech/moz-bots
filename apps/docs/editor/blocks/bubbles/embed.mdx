El bloque de burbuja de incrustación te permite mostrar un sitio web o un iframe a tu usuario. Puedes pegar una URL de video de muchas fuentes, incluyendo YouTube, Vimeo y mp4.

<Info>
  No todos los sitios web permiten la incrustación. Si ves un espacio en blanco,
  significa que el sitio web que estás intentando incrustar no lo permite.
</Info>

## Incrustar un PDF alojado en Google Drive

Para esto, necesitarás seleccionar el archivo PDF que deseas incrustar. Haz clic derecho > Vista previa > Más acciones > Abrir en una nueva ventana. Ahora haz clic en Más acciones > Incrustar elemento.

Copia el código de incrustación y pégalo en la configuración del bloque de burbuja de incrustación.

## Esperar evento

Habilita esto si eres el propietario del sitio web que deseas incrustar y te gustaría continuar el flujo del bot solo cuando se envíe un evento desde la incrustación al bot. Esta ejecución del evento necesita ser realizada en el sitio web incrustado. Aquí hay un ejemplo:

```js
window.parent.postMessage(
  {
    name: 'Mi evento',
    data: 'Datos personalizados pasados a la variable del mozbot',
  },
  '*'
)
```
