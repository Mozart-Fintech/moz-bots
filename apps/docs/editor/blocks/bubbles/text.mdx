## Insertar un enlace

Puedes insertar un enlace en tus burbujas de texto usando el ícono de enlace en el editor:

<Frame style={{ maxWidth: '400px' }}>
  <img
    src="/images/blocks/bubbles/text-link.png"
    alt="Ícono de enlace de texto"
  />
</Frame>

Puedes insertar cualquier enlace que comience con `http:`, `https:`, `mailto:`, `tel:` o `sms:`. También puede ser una variable.
