## Formato

La configuración de `Formato` te permite personalizar el formato de la fecha seleccionada. En el fondo, se realiza utilizando la biblioteca [date-fns](https://date-fns.org/). Puedes usar cualquiera de los [tokens de formato](https://date-fns.org/docs/format) soportados por la biblioteca.

Aquí hay algunos ejemplos:

```text
yyyy-MM-dd
yyyy-MM-dd HH:mm:ss
dd/MM/yy
dd/MM/yyyy HH:mm:ss
d.MM.yy
```
