## Ícono personalizado

Para insertar un ícono personalizado, necesitarás insertar contenido SVG. Debe comenzar con `<svg>` y terminar con `</svg>`. Puedes encontrar excelentes íconos de código abierto en [Feather](https://feathericons.com/)
