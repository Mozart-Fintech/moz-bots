El bloque de entrada de Sitio web te permite pedirle a tu usuario una URL. Verificará si está correctamente formateada.

El mensaje de reintento se mostrará cada vez que Mozbot detecte que la URL no está correctamente formateada.

No verificará si la dirección de correo electrónico es **válida**. Para hacer eso, tendrás que activar un [bloque Webhook](/editor/blocks/integrations/webhook) y llamar a una API de servicio de validación de correos electrónicos.
