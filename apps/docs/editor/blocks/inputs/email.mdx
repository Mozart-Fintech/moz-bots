El bloque de entrada de Correo Electrónico te permite pedirle a tu usuario un correo electrónico. Verificará si está correctamente formateado.

El mensaje de reintento se mostrará siempre que Mozbot detecte que el correo electrónico no está correctamente formateado.

No verificará si la dirección de correo electrónico es **válida**. Para hacer eso, tendrás que activar un [bloque de Webhook](/editor/blocks/integrations/webhook) y llamar a una API de servicio de validación de correo electrónico.
