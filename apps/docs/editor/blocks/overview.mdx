---
title: Descripción General
---

Los bloques son los componentes atómicos de un chat de mozbot. Puedes encadenar cualquier bloque para crear flujos de conversación complejos. Cuando arrastras y sueltas un bloque en el flujo de mozbot, por defecto se insertará en un grupo que puede tener un título. Los bloques siempre están contenidos en grupos.

Hay múltiples categorías de bloques para facilitar la búsqueda del bloque adecuado para tus necesidades:

## Burbujas

Los bloques de burbujas se utilizan para mostrar burbujas que pueden usarse para mostrar información al usuario.

## Entradas

Los bloques de entrada se utilizan para pedir al usuario una entrada. Detendrán la conversación hasta que el usuario proporcione una entrada.

<Frame style={{ maxWidth: '400px' }}>
  <img src="/images/blocks/inputs.png" alt="Bloques de entrada" />
</Frame>

## Lógica

Los bloques de lógica se utilizan para realizar operaciones en segundo plano. No son visibles para los usuarios.

  <img src="/images/blocks/logic.png" alt="Bloques de lógica" />
</Frame>

## Integraciones

Los bloques de integración se utilizan para activar operaciones de servicios externos.
