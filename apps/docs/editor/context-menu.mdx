---
title: Menú contextual
icon: bars
---

Puedes hacer clic derecho en diferentes elementos en el constructor para activar un menú con diferentes acciones.

Si haces clic derecho en un grupo, verás un menú con las siguientes opciones:

<Frame>
  <img src="/images/group-context-menu.png" alt="Menú contextual" />
</Frame>

- **Duplicar** para crear un grupo duplicado con los mismos bloques y configuraciones
- **Eliminar** para eliminar el grupo

Si haces clic derecho en un bloque, verás un menú con las siguientes opciones:

- **Duplicar** para crear un bloque duplicado debajo del bloque duplicado con las mismas configuraciones
- **Eliminar** para eliminar el bloque

Si haces clic derecho en un borde, verás un menú con las siguientes opciones:

<Frame>
  <img src="/images/edge-context-menu.png" alt="Menú contextual" />
</Frame>

- **Eliminar** para eliminar el borde
