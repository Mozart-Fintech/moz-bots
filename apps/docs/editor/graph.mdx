El Gráfico es donde organizas el flujo de conversación y conectas los [bloques](./blocks/overview) del Mozbot.

## Gestos

En las preferencias del usuario, en la sección `Gestos del Gráfico`, puedes elegir entre gestos de `Ratón` y `Trackpad`.

### Ratón

**Seleccionar**: `Shift` + arrastrar con `clic izquierdo`

**Zoom**: `Ctrl` + rueda del ratón

**Desplazar**: arrastrar con `clic izquierdo` o rueda del ratón para desplazamiento vertical y `Shift` + rueda del ratón para desplazamiento horizontal

### Trackpad

**Seleccionar**: `Clic` + arrastrar

**Zoom**: Pellizcar

**Desplazar**: Usar dos dedos
