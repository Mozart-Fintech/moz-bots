title: Preguntas Frecuentes
---

## ¿Por qué algunos de mis resultados están vacíos?

Tu mozbot recopila resultados tan pronto como tu usuario responde la primera entrada. Si otras entradas tienen una respuesta en blanco, significa que el usuario nunca las respondió y abandonó el bot.

## ¿Cómo puedo eliminar un bloque o una conexión?

Puedes hacer clic derecho en la mayoría de los elementos del gráfico para abrir un menú contextual.

## ¿Está disponible la toma de control humana en Mozbot?

El chat en vivo no está integrado de forma nativa. Mozbot está específicamente diseñado para chats asincrónicos. Aún puedes tener una función de toma de control humana utilizando el [bloque de Chatwoot](/editor/blocks/integrations/chatwoot). Este abrirá una caja de chat en vivo de Chatwoot cada vez que se active el bloque.

También puedes implementar un excelente soporte al cliente con Mozbot utilizando un sistema de tickets como [Freshdesk](https://freshdesk.com/) o [Zendesk](https://www.zendesk.com/). Puedes crear un ticket cada vez que tu usuario responda al bot.

## ¿Hay un programa de afiliados disponible?

No planeo tener un programa de afiliación para Mozbot.

Como cliente, no me gusta cuando alguien promociona una herramienta con una afiliación porque podríamos pensar que la única razón por la que la está promocionando es que puede ganar mucho en comparación con otras herramientas.

Las recomendaciones naturales resuenan mucho más conmigo. Mi objetivo es crear la mejor experiencia de usuario posible, así que realmente espero que recomiendes Mozbot de forma gratuita. 🤝

## ¿Cómo puedo cambiar mi contraseña?

Mozbot no almacena ninguna contraseña. Su inicio de sesión funciona con Github, Google, Facebook y enlace mágico por correo electrónico.

## ¿Puedo establecer una variable con el bloque de Script?

No, el bloque de script solo está destinado a ejecutar un script. No puedes establecer una variable con/en él. Si necesitas establecer una variable con algún código, puedes usar el [bloque de Establecer variable](/editor/blocks/logic/set-variable).
