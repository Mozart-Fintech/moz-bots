## Mozbot v6

- Las variables de lista ahora no muestran automáticamente el último elemento cuando se insertan en una burbuja. Era demasiado "mágico". Ahora puedes utilizar la función de código en línea para obtener fácilmente el último elemento de una lista:

  ```
  {{={{List var}}.at(-1)=}}
  ```

  Consulta la nueva [sección de formato de variables en línea](./editor/variables) para obtener más información.

- El prefijo de entrada ahora está deshabilitado por defecto. Aún puedes habilitarlo en la pestaña [Configuración](./settings/overview) de tu bot.
