---
title: Fragmento de inserción de script
---

La opción de inserción de script es útil solo si no tienes acceso al árbol HTML de tu aplicación o si tu creador de sitios web solo te permite insertar fragmentos de script en línea.

De lo contrario, es preferible seguir las instrucciones de inserción de [HTML y Javascript](./libraries/html-javascript) porque los fragmentos de script son solo scripts que inyectarán el código del método de inserción de HTML y Javascript.
