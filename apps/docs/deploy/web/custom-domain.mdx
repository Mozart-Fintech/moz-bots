---
title: Añadir un dominio personalizado
---

Puedes vincular un dominio personalizado a tu mozbot en la pestaña "Compartir".

<Frame>
  <img
    src="/images/custom-domain/add-domain.png"
    alt="Botón Añadir mi dominio"
  />
</Frame>

Para conectar un nuevo dominio, sigue las instrucciones:

<Frame>
  <img
    src="/images/custom-domain/instructions.png"
    alt="Instrucciones para añadir dominio"
  />
</Frame>

Una vez que hayas añadido el registro DNS correspondiente, haz clic en el botón "Guardar". Es posible que tengas que esperar unos minutos antes de que el registro se propague correctamente.

## Solución de problemas

Si tu dominio no está configurado o verificado correctamente, verás este icono de error junto al enlace de tu dominio:

<Frame>
  <img src="/images/custom-domain/invalid-config.png" alt="Icono de error" />
</Frame>

Asegúrate de hacer clic en él para ver qué se requiere hacer para solucionar el problema.
