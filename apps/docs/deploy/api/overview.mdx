Tu bot también puede ejecutarse con solicitudes HTTP. Esto es útil si deseas integrar tu bot con otro servicio o si deseas usarlo en un lenguaje de programación diferente.

## Prueba

1. Todas tus solicitudes deben estar autenticadas con un token de API. [Ver instrucciones](/api-reference/authentication).
2. Para iniciar el chat, envía una solicitud POST a `https://mozbot.mozartfintech.com/api/v1/mozbots/<mozbot ID>/preview/startChat`. La primera respuesta contendrá un sessionId que necesitarás para solicitudes posteriores.
3. Para enviar respuestas, envía solicitudes POST a `https://mozbot.mozartfintech.com/api/v1/sessions/<session ID>/continueChat`
   con el siguiente cuerpo JSON:

   ```json
   {
     "message": "Esta es mi respuesta"
   }
   ```

Consulta la [Referencia de la API para iniciar el chat de vista previa](/api-reference/chat/start-preview-chat) para más información.
