## Limitaciones

El entorno de WhatsApp tiene algunas limitaciones que debes tener en cuenta al construir el bot:

- Los archivos de imagen GIF y SVG no son compatibles. No se mostrarán.
- Solo se admiten videos en formato .mp4 (Consulta [Tipos de medios compatibles](https://developers.facebook.com/docs/whatsapp/cloud-api/reference/media#supported-media-types) para más información).
- El contenido de los botones no puede tener más de 20 caracteres. Si el contenido es más largo, se truncará.
- WhatsApp solo permite mostrar 3 botones a la vez. Así que trabajamos alrededor de eso añadiendo mensajes de "..." para mostrar más botones.
- Los bloques incompatibles, si están presentes, serán omitidos:

  - Bloque de entrada de pago
  - Bloque de Chatwoot
  - Bloque de script
  - Bloque de Google Analytics
  - Bloques de Meta Pixel
  - Bloque de Cal.com

### Configuración

Puedes personalizar cómo se comporta tu bot en WhatsApp en la sección `Configurar integración`

<Frame>
  <img
    src="/images/whatsapp/configure-integration.png"
    alt="Configurar integración de WhatsApp"
  />
</Frame>

**Tiempo de expiración de la sesión**: Un número de 0 a 48 que es la cantidad de horas después de las cuales la sesión expirará. Si el usuario no interactúa con el bot por más tiempo que el tiempo de espera, la sesión expirará y si el usuario envía un nuevo mensaje, comenzará un nuevo chat. El valor predeterminado es 4 horas.

**Condición de inicio del bot**: Una condición que se evaluará cuando un usuario inicie una conversación con tu bot. Si la condición no se cumple, el bot no se activará.

## Preguntas frecuentes

<AccordionGroup>
  <Accordion title="¿Cuántos números de WhatsApp puedo usar?">
    Puedes integrar tantos números como desees. Ten en cuenta que Mozbot no proporciona esos números. Trabajamos como una "Trae tu propia aplicación de Meta" y te damos instrucciones claras sobre [cómo configurar tu aplicación de Meta](./create-meta-app).
  </Accordion>
  <Accordion title="¿Puedo vincular múltiples bots al mismo número de WhatsApp?">
    Sí, puedes. Tendrás que añadir una "Condición de inicio del bot" a cada uno de tus bots para asegurarte de que el bot correcto se active cuando un usuario inicie una conversación.
  </Accordion>
  <Accordion title="¿La integración con WhatsApp requiere alguna API de pago?">
    Integras tus mozbots con tu propia Plataforma de Negocios de WhatsApp, que es el servicio oficial de Meta. En este momento, las primeras 1,000 conversaciones de servicio cada mes son gratuitas. Para más información, consulta <a href="https://developers.facebook.com/docs/whatsapp/cloud-api/get-started#pricing---payment-methods" target="_blank">su documentación</a>
  </Accordion>
  <Accordion title="Recibo el mensaje inicial de vista previa pero luego el bot no responde">
    Esto puede deberse a una discrepancia entre el número de WhatsApp que se muestra en la configuración del perfil de tu aplicación de WhatsApp y el número que has escrito en Mozbot.

    Por ejemplo, algunos brasileños tienen 2 números, 1 con 8 dígitos y el otro con 9 dígitos. En Mozbot, necesitas escribir el número que se muestra en la configuración del perfil de tu aplicación de WhatsApp para asegurarte de que el bot funcione correctamente.

  </Accordion>
</AccordionGroup>
