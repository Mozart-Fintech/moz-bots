---
title: Avatares dinámicos
---

Puedes decidir cambiar el avatar del anfitrión dinámicamente para que tu bot pueda mostrar múltiples avatares. Aquí tienes un ejemplo simple:

<Frame>
  <img
    src="/images/guides/multi-avatars/demo.gif"
    alt="Demostración de múltiples avatares"
  />
</Frame>

Para esto, dirígete a la pestaña "Tema" y configura el avatar del anfitrión a una URL variable:

<Frame>
  <img
    src="/images/guides/multi-avatars/bot-variable.png"
    alt="Bot con URL variable"
  />
</Frame>

Luego, puedes establecer esta variable a cualquier URL en tu flujo para que el avatar del anfitrión cambie dinámicamente.

<Frame>
  <img
    src="/images/guides/multi-avatars/flow-example.png"
    alt="Ejemplo de flujo con múltiples avatares"
  />
</Frame>

El primer bloque define todos los avatares que se pueden mostrar. El segundo bloque comienza configurando el avatar al avatar del mago. El bloque final comienza configurando el avatar del herrero.
